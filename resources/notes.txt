# Android
- to setup just look at the dockerfile in cargo-apk
- i had to use the github version of cargo-apk, not the cargo install one.
- android build environment guide (old): https://macroquad.rs/articles/android/#a-manual-way
- package with docker: docker run --rm -v $(pwd)":/root/src" -w /root/src notfl3/cargo-apk cargo quad-apk build --release
- icon sizes:
```
mipmap-mdpi 	48×48 px 	MDPI (baseline)
mipmap-hdpi 	72×72 px 	HDPI (1.5×)
mipmap-xhdpi 	96×96 px 	XHDPI (2×)
mipmap-xxhdpi 	144×144 px 	XXHDPI (3×)
mipmap-xxxhdpi 	192×192 px 	XXXHDPI (4×)
```
